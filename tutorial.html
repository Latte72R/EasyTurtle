使い方

基本説明

ウィジェットを追加する

右側のリストでウィジェットを選択すると左側のリストにウィジェットが追加されます。

ウィジェットを変更する

ウィジェットを右クリックすることでメニューが表示されます。

　　名称　　 　　　　　　　　説明　　　　　　　　
Top ウィジェットを一番上に移動する
Bottom ウィジェットを一番下に移動する
Up ウィジェットを一個上に移動する
Down ウィジェットを一個下に移動する
Copy ウィジェットをコピーする
Cut ウィジェットをカットする
Delete ウィジェットを削除する
Option オプションを表示する　　　　　　※１
Enable ウィジェットを有効化する　　　　※２
Disable ウィジェットを無効化する　　　　※２
⇧Paste⇧ ウィジェットを上にペーストする　※３
⇩Paste⇩ ウィジェットを下にペーストする　※３

※注：WriteウィジェットやColorウィジェットにはオプションもついています。

複数ウィジェットのコピーとペースト

コピー

Copy Selectedを押すことでコピーすることができます。

ペースト

Paste Widgetsを押すことで左側のリストにコピーしたウィジェットを追加できます。

実行する

Run Programを押すと新しいウィンドウが作られ実行されます。

データを保存する

Save Programを押すと保存用ウィンドウが作られファイル名を入れると保存できます。

保存したデータを開く

Open Programを押すと開く用ウィンドウが作られファイルを選択すると開くことができます。
